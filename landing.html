Mis Proyectos y Soluciones

Explora la convergencia entre la ingeniería de control y el desarrollo de software. Aquí detallo cómo utilizo lenguajes como Java, herramientas de IoT y metodologías ágiles para resolver desafíos técnicos y optimizar procesos en entornos reales.

Tinta Nómada | E-commerce

Estrategia Digital y Gestión Ágil de Proyectos

Colaboré en la transformación digital de un negocio de productos personalizados mediante la digitalización de pedidos y la creación de canales de atención eficientes. El proyecto fue gestionado bajo la metodología SCRUM, rotando roles y aplicando tableros Kanban para optimizar el flujo de trabajo. Esta experiencia demuestra mi capacidad para alinear necesidades de negocio con soluciones tecnológicas estructuradas.

Tecnologías: JavaScript, HTML, CSS, Bootstrap, Java, MySQL, Spring Boot.

Optimización de Cultivos IoT | Ingeniería

Monitoreo Inteligente y Automatización de Datos

Como proyecto de titulación en el IPN, desarrollé un sistema integral para el cultivo de setas rosas (Pleurotus djamor). Implementé algoritmos de control para variables ambientales utilizando microcontroladores conectados a internet. El sistema consume la API de Google Sheets para el almacenamiento masivo de datos y la API de Telegram para notificaciones de estado. Impacto: Logré optimizar la producción en un 18.6% y reduje los tiempos de respuesta del sistema en un 14.3%.

Tecnologías: C, C++, HTML, Sistemas embebidos.

CRUD Java MySQL | Mantenedor de Usuarios

Persistencia de Datos y Arquitectura con JDBC

Desarrollo de backend robusta centrada en la gestión de entidades de usuario. Utilicé Java SE con una arquitectura orientada a objetos para implementar un sistema CRUD (Create, Read, Update, Delete) completo. La persistencia se gestiona mediante una base de datos relacional MySQL, conectada a través de la API JDBC, asegurando la integridad de la información y un manejo eficiente de las consultas SQL.

Tecnologías: Java, MySQL.
